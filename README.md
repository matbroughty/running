Personal training plan and log for the **Faversham 10K** on **Sunday 27 September 2026**.
Goal: **sub-45:00** (4:30 / km).

This repo lives inside an Obsidian vault, so files use `[[wikilinks]]` and render nicely in Obsidian's reading view. They also render fine on GitHub as plain Markdown.

## Plan structure

| Block         | Weeks     | Focus                                         |
| ------------- | --------- | --------------------------------------------- |
| Base          | 1 – 4     | Aerobic base, build weekly rhythm             |
| Threshold     | 5 – 7     | Sustained faster work                         |
| Recovery      | 8         | Step the load back, absorb                    |
| Race specific | 9 – 13    | 10K-pace intervals, longer tempo              |
| Sharpening    | 14 – 16   | Race-pace volume, intensity holds, load drops |
| Taper         | Race week | Stay sharp, race fresh                        |

## How to use this repo

- Tick checkboxes in [`Faversham-10K-2026.md`](./Faversham-10K-2026.md) as sessions are completed.
- Fill in **Actuals**, **Weight**, and the **Retro** at the end of each week.
- Add new sessions to the top of the **Sessions** list in [`Training-Log.md`](./Training-Log.md) using the template at the bottom of that file.
- Update [`Dashboard.md`](./Dashboard.md) when current metrics change (weight, Parkrun PB, long run distance).
- Weigh in every Sunday morning and record in the weekly weigh-in table.
